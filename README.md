### STRUCTURE FOR ORACLE BANK MIGRATIONS

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Arquive TypeORM

# Important
change the **ormconfig.example.json** file
to **ormconfig.json**

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

```bash
# migrations create
$ npm run migration:create -- -n "NAME MIGRATIONS"

# migration run
$ npm run migration:up

# migration revert
$ migration:down
```
